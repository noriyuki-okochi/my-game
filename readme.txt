〇ゲーム方法
	1.「playing mode」を選択する。

	2.<ctrl+s>｜<left-bottom click>｜<enter>でプレイを開始する。

	3.「操作面の指定」、「回転方向」をキー入力する。

	　（操作面の指定）
	　　1)操作面の色による指定方法（絶対的指定）
	　　　　'G'/'B'/'O'/'R'/'Y'/'W'
	    2)初期位置の基準座標に対する方角による指定方法（絶対的指定）
	       <ctrl+m>で絶対入力モード(abs)に設定する
	　　　　's'/'n'/'e'/'w'/'t'/'b'
	    3)カメラ視点での基準座標に対する方角による指定方法（相対的指定）
 	       <ctrl+m>で相対入力モード(rel)に設定する
	　　　	's'/'n'/'e'/'w'/'t'/'b'

	　（回転方向の指定）
	　　1)右90度回転（基本動作）
	　　　　';'
	　　2)左90度回転（基本動作）
	　　　　'-'
	　（複合操作の指定）
	    1)Y交換子
	　　　<面指定1>,'y',<面指定2>,<指定1面回転方向>
	    2)Z交換子
	　　　<面指定1>,'z',<面指定2>,<指定1面回転方向>
	    3)中抜き
	　　　<面指定>,'v',<回転方向>
	    4)東周り世界一周
	　　　<始点面指定>,'.',<終点面指定>,<始点面回転方向>
	    5)西周り世界一周
	　　　<始点面指定>,',',<終点面指定>,<始点面回転方向>

	4)操作の取り消し
	　（直前操作の逆操作実行）
	　　　'u'
	　（操作ログ表示の操作キャンセル）
	　　　<ctrl+z>

	5)コマンドラインによる基本操作実行
	　（コマンドライン入力の開始）
	　　　<ctrl+i>
	　（コマンドライン入力）
	      {<操作面の絶対指定>,<基本動作>,}...
	          <操作面の絶対指定>::'S'/'N'/'E'/'W'/'T'/'B'
	           (*)基準座標に対する方角をupper-caseで入力
	　（操作ログの表示コマンドをコピー）
	　　　<ctrl+c>
	           (*)表示コマンドは基準座標に対する方角をupper-caseで表示
	　（操作ログのコピーコマンドをコマンドラインへ複写）
	　　　<ctrl+p>
	　（コマンドライン編集）
	      <delete>   :全削除
	      <bs>       :カーソル（文字入力位置）の前一1文字削除
	      <左矢印>   :カーソルを一文字前へ移動
	      <右矢印>   :カーソルを一文字後へ移動

	6)操作の確定（confirm）
	　（操作ログ表示の操作確定）
	      <enter>

	7)カメラ視点の移動
	      <右矢印リピート>:右側へ移動
	      <左矢印リピート>:左側へ移動
	      <上矢印リピート>:上側へ移動
	      <下矢印リピート>:下側へ移動
	      <home>  :初期位置（右上45度）へ移動
	      'o'     :現在位置の反対側へ移動
	      '+'     :ズームアップ

	8)キューブ自体の回転
	      <pg up> :Pitch方向（X軸中心）90度手前へ回転
	      <pg dn> :Pitch方向（X軸中心）90度背面へ回転
	      <scroll>:Rpll方向（Y軸中心）90度左へ回転
	      <pause> :Rpll方向（Y軸中心）90度右へ回転
	      <numlock>:初期位置に戻す

〇セッティングモードのコマンドライン入力
	1.「settin mode」を選択する。
	2.ガイダンスメニュー、メッセージに従って、コマンドラインにコマンドをキー入力する。
	　
	(set)	：パラメータ設定メニューを表示する
		  cmd. file name: デフォルトの操作コマンドファイル名。
		  suto reg.     : confirmの実行で現在のパターン情報をファイルに保存する。
		  auto search   : confirmの実行で現在のパターン情報がDBに登録済か検索する。
		  auto interval : 検索した解法の一操作を自動実行する間隔（秒単位）

	(cl)	：ガイダンスメッセージ、コマンドラインのクリア
	(reg)	：現在のパターン情報をファイルに保存する
		  reg <file-path> 

	(ld)	：ファイルからパターン情報を復元して表示する
	  	  ld <file-path>
 
	(cmp)	：現在のパターン情報を完成パターンとしてDBのcomp_patternテーブルに登録する
	  	  cmp r

	(lw)	：操作ログファイルを別ファイルにコピーする
	  	  lw <file-path> [[<from-line>]'-'[<to-line>]]

	(pt r)	：操作ログファイルの先頭から完成(completed)までの確定(comfirm)により記録
	　　　　　されたパターン情報、操作コマンドをDBのpattern/solutionテーブルに登録する。
	  	  pt r [<log-file-path>]
	　　　       <log-file-path>のデフォルト::現在の操作ログファイル

	(pt d)	：指定パターンIDをpattern/solutionテーブルから削除する。
	  	  pt d  <pattern-id>
	　　　       <pattern-id>の部分一致で検索する


	(pt s)	：現在表示のパターン情報と一致するパターンIDをpatternテーブルから検索する。
	  	  pt s

	(pt v)	：指定パターンIDに一致するパターン情報をpatternテーブルから検索して、復元
	　　　　　表示する。
	  	  pt v  <pattern-id>
	　　　       <pattern-id>の部分一致で検索する

	(pt g)	：指定パターンIDの解法をsolutionテーブルから検索する。
	  	  pt g [<pattern-id>]  [-e [-s]]
	　　　       <pattern-id>のデフォルト::現在表示中のパターン
		     -eオプション::検索結果の解法を実行する
		     -sオプション::基本操作をステップ実行する

〇コマンドラインのキー入力
	・入力は挿入モードのみサポート。
	・入力位置はカーソル’＿’で表示される。
	・入力文字はカーソル位置に挿入され、カーソルは１文字右に移動する。
	<delete> ：入力文字の全削除
	<bs>     ：カーソルの左１文字の削除
	<左矢印> ：カーソルを左へ１文字移動
	<右矢印> ：カーソルを右へ１文字移動
